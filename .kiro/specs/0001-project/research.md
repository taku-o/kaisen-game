# 海戦ゲーム（Battleship）調査レポート

## 1. 海戦ゲームとは

海戦ゲーム（Battleship、バトルシップ）は、2人用の戦略的推理ゲームです。各プレイヤーは方眼紙やボード上に自分の艦隊を秘密裏に配置し、交互に相手の艦の位置を推理しながら攻撃を行います。相手の艦隊を全て撃沈したプレイヤーが勝利します。

別名として「戦艦ゲーム」「軍艦ゲーム」「潜水艦ゲーム」などとも呼ばれます。

## 2. 歴史と起源

### 起源
- **1900年頃**：Clifford Von Wickleによって開発されたとされる
- **第一次世界大戦期**：紙とペンで遊ぶゲームとして広く知られるようになった
- **1907年**：ロシアの詩人リューリク・イヴネフの日記に、このゲームで遊んだ記録が残っている
- フランスのゲーム「L'Attaque」や、1890年のE.I. Horsemanのゲーム「Baslinda」との関連も指摘されている
- ロシアの将校たちが第一次世界大戦前にプレイしていたとも言われる

### 商業化の歴史
| 年 | 出来事 |
|----|--------|
| 1931年 | Starex社が「Salvo」として米国で最初の商業版を発売 |
| 1930-40年代 | 各社から様々なバージョンが発売（Combat: The Battleship Game等） |
| 1943年 | 「Broadsides, the Game of Naval Strategy」が発売 |
| 1967年 | Milton Bradley社がプラスチック製ペグボードと船のミニチュアを使用したボードゲーム版を発売 |
| 1977年 | 電子版バトルシップが登場 |
| 1985年 | CMで有名なフレーズ「You sank my battleship!」が誕生 |
| 2025年 | National Toy Hall of Fame（国立おもちゃの殿堂）に選出 |

現在、29言語に翻訳され、40か国以上で販売、累計1億個以上を売り上げている。

## 3. 古典的なルール（紙とペン版）

### 3.1 必要なもの
- 紙（方眼紙が望ましい）
- ペンまたは鉛筆
- 2名のプレイヤー

### 3.2 グリッドの準備
各プレイヤーは2つのグリッド（方眼）を用意します：

1. **自陣グリッド（Ocean Grid）**：自分の艦を配置するグリッド
2. **攻撃記録グリッド（Targeting Grid）**：相手への攻撃結果を記録するグリッド

標準的なグリッドサイズは**10×10マス**です。
- 縦軸：A〜Jのアルファベット
- 横軸：1〜10の数字

### 3.3 艦の種類とサイズ

標準的な艦隊構成（Milton Bradley版）：

| 艦種 | マス数 | 記号例 |
|------|--------|--------|
| 空母（Aircraft Carrier） | 5 | A |
| 戦艦（Battleship） | 4 | B |
| 巡洋艦（Cruiser） | 3 | C |
| 潜水艦（Submarine） | 3 | S |
| 駆逐艦（Destroyer） | 2 | D |

**合計：5隻、17マス**

※ドイツ版では10隻（空母1、戦艦2、駆逐艦3、巡洋船4）を使用する等、バリエーションがあります。

### 3.4 艦の配置ルール

1. **配置方向**：艦は水平または垂直に配置する（**斜めは不可**）
2. **グリッド内配置**：艦はグリッドの境界線を超えてはならない
3. **重複禁止**：艦同士が同じマスを共有することはできない
4. **隣接ルール**（オプション）：ドイツ版等では、艦と艦の間に最低1マスの間隔が必要（角での接触は可）
5. **配置の秘密**：相手に自分の配置を見せてはならない
6. **移動禁止**：ゲーム開始後、艦を移動することはできない

## 4. ゲームの流れ

### 4.1 ゲーム開始
1. 両プレイヤーが自陣グリッドに艦を配置する
2. じゃんけん等で先攻を決定する

### 4.2 ターンの進行
プレイヤーは交互に以下を行います：

#### 攻撃フェーズ
1. **座標の宣言**：攻撃するマスの座標を宣言する（例：「B3」）
2. **結果の応答**：相手は攻撃されたマスを確認し、結果を報告する
   - **ミス（Miss / Water）**：そのマスに艦がない場合
   - **命中（Hit）**：そのマスに艦がある場合
3. **記録**：
   - 攻撃側：攻撃記録グリッドに結果をマーク（命中=X、ミス=O等）
   - 防御側：命中した場合、自陣グリッドにマーク

#### 撃沈の報告
艦の全マスが命中した場合、防御側は「（艦名）撃沈！」と報告する。
- 例：「You sank my battleship!」（戦艦撃沈！）

### 4.3 ターン交代
攻撃と応答が完了したら、相手プレイヤーのターンに移る。

## 5. 勝利条件

**相手の艦隊を全て撃沈したプレイヤーが勝利**

すなわち、相手の全ての艦（5隻、17マス）に命中させた時点で勝利が確定します。

## 6. バリエーションとハウスルール

### 6.1 Salvo（サルボ）モード
- 1ターンに複数回攻撃できる
- **攻撃回数 = 残存艦の数**
- 艦が沈むごとに攻撃回数が減少する
- より戦略的でスピーディな展開になる

### 6.2 インド版バリアント
- 異なる艦隊構成を使用
- 命中した場合、具体的な位置ではなく「命中数」のみを報告
- より高度な推理が必要

### 6.3 移動ルール（古典的バリアント）
一部のルールでは、攻撃の代わりに艦を移動させることができる：
- 1ターンに攻撃か移動のいずれかを選択
- 移動は1マス、または艦の長さ分まで可能（ルールによる）

### 6.4 隣接禁止ルール
艦同士を隣接させて配置できないルール（ドイツ版等）：
- 艦と艦の間には最低1マスの空白が必要
- 角同士の接触は許可される場合もある

### 6.5 簡易版（5×5グリッド）
- 小さなグリッドと少ない艦数で遊ぶ
- 子供向けや短時間プレイに適している
- 艦構成例：戦艦(3マス)、駆逐艦(2マス)、潜水艦(1マス)

### 6.6 連続攻撃ルール
- 命中した場合、追加でもう1回攻撃できる
- ミスするまで攻撃を続行できる
